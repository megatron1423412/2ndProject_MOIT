# 자동건조

냉방 중 실내기 내부에는 결로로 물기가 생긴다. 운전 종료 뒤 내부가 젖은 채로 남으면 냄새와 곰팡이 발생 가능성이 커질 수 있다.

자동건조는 전원을 끈 뒤 일정 시간 송풍을 작동해 내부 수분을 줄이는 기능이다.

## 확인할 점

- 자동건조 지원 여부
- 종료 후 작동 시간
- 사용자가 중단할 수 있는지
- 벽걸이형에서도 지원하는지

자동건조가 없는 제품은 사용 후 송풍을 별도로 작동하는 관리가 필요할 수 있다.

# 스마트폰 연결

스마트폰 제어는 실내에서 리모컨을 대체하기 위한 기능이라기보다 외부 원격 운전과 사용 정보 확인에 더 유용할 수 있다.

대표적인 활용:

- 귀가 전에 미리 냉방 시작
- 현재 운전 상태 확인
- 소비전력이나 사용량 확인
- 예약과 자동화 설정

앱 설치, 계정, 무선 네트워크가 필요할 수 있으므로 지원 조건을 확인한다.

# 풍향조절

제품마다 자동으로 움직일 수 있는 방향이 다르다.

- 상하 자동 스윙
- 좌우 자동 스윙
- 상하·좌우 모두 자동
- 한 방향은 자동, 다른 방향은 수동 고정

특히 벽걸이형은 상하만 자동이고 좌우는 수동인 제품이 많을 수 있다.

소파, 침대, 식탁 위치 때문에 특정 방향 조절이 중요하다면 `풍향조절 지원`이라는 포괄적인 문구가 아니라 자동 조절 가능한 실제 축을 확인한다.
