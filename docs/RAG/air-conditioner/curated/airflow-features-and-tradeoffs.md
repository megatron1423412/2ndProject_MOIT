# 바람 기능을 마케팅 명칭보다 목적별로 보기

제조사마다 서로 다른 명칭을 사용하지만 구매 목적은 대체로 다음과 같이 나뉜다.

- 몸에 직접 닿는 찬바람을 줄이고 싶음
- 방을 빠르게 식히고 싶음
- 멀리 떨어진 공간까지 바람을 보내고 싶음

기능 이름보다 사용 목적과 실제 대안을 함께 비교한다.

# 간접바람

간접바람 기능은 찬바람이 몸에 직접 닿는 불편을 줄이는 데 목적이 있다.

## 유용한 사용자

- 침실에서 직접 바람을 피하고 싶은 사람
- 소파나 침대 위치상 송풍구와 마주 보는 사람
- 강한 바람에 민감한 사람

## 대안

전용 기능이 없어도 다음으로 비슷한 목적을 달성할 수 있다.

- 풍향을 사람과 다른 방향으로 조절
- 풍량을 낮춤
- 절전 또는 저출력 모드 사용
- 가구 배치나 설치 위치 조정

따라서 간접바람은 중요한 편의 기능일 수 있지만 모든 사용자에게 최우선 구매 기준은 아니다.

# 급속냉방

급속냉방은 에어컨을 강한 출력으로 운전해 목표 온도에 빨리 도달하도록 하는 모드다.

빠른 냉방이 중요하다면 기능 이름만 비교하기보다 다음을 먼저 본다.

- 공간에 충분한 냉방능력
- 단열과 일사량
- 스마트폰 원격제어로 미리 가동 가능 여부

용량이 부족한 제품에 급속냉방 기능이 있다고 해서 근본적인 냉방부하 문제가 해결되는 것은 아니다.

# 원거리냉방

원거리냉방은 팬과 기류 설계로 바람을 멀리 보내는 기능이다.

멀리 보내는 힘을 높이면 소음이 커질 수 있고, 소음을 낮추면 체감 효과가 줄 수 있다. 제조사 간 객관적인 비교 자료가 부족하면 우선순위를 높이기 어렵다.

넓고 긴 공간에서는 전용 원거리 기능 외에 서큘레이터를 함께 사용하는 대안도 비교한다.

# 구매 판단

바람 기능은 체감 선호가 큰 영역이다. 사용자가 직접 바람, 소음, 냉방 속도 중 무엇을 더 중요하게 생각하는지 확인한 뒤 추천한다.
