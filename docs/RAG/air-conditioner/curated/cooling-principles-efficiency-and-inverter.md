# 에어컨이 열을 이동시키는 방식

에어컨은 실내기와 실외기가 냉매를 순환시키며 실내의 열을 외부로 이동시킨다.

- 실내기에서는 냉매가 기화하면서 주변의 열을 흡수하고, 팬이 차가워진 공기를 실내로 보낸다.
- 실외기에서는 열을 흡수한 냉매가 응축되며 외부로 열을 방출한다.

전력 사용의 큰 부분은 실외기 운전과 관련된다. 실내 온도가 설정 온도보다 높으면 실외기가 적극적으로 작동하고, 목표 온도에 가까워지면 인버터 제품은 출력을 낮출 수 있다.

# 냉방능력, 소비전력, 냉방효율

## 냉방능력

얼마나 큰 공간을 냉방할 수 있는지와 관련된 성능이다.

## 소비전력

제품이 작동할 때 사용하는 전력이다. 인버터 제품은 상황에 따라 소비전력이 범위로 표시되기도 한다.

## 냉방효율

같은 냉방효과를 내기 위해 얼마나 적은 전력을 쓰는지를 비교하는 지표다. 효율이 높을수록 같은 조건에서 적은 전력으로 더 큰 냉방효과를 낼 가능성이 높다.

# 표기값만으로 냉방효율을 직접 계산하지 않기

제품표의 정격 냉방능력과 정격 소비전력을 단순히 나눈 값이 공식 냉방효율과 항상 같지는 않다. 공식 효율은 표준 시험 환경에서 실제로 측정된 냉방량과 전력 사용량을 기준으로 산정되기 때문이다.

따라서 서로 다른 제품을 비교할 때 임의 계산값보다 공인된 효율 지표와 동일 기준의 시험값을 사용한다.

# 인버터 방식

인버터는 모터와 압축기의 출력을 상황에 맞게 조절한다. 목표 온도에 도달하기 전에는 강하게 작동하고, 도달한 뒤에는 필요한 수준으로 출력을 낮춰 온도를 유지한다.

장점은 다음과 같다.

- 실내 온도 변동이 상대적으로 작음
- 목표 온도 이후 저출력 유지 가능
- 잦은 완전 정지와 재가동을 줄일 수 있음
- 적절한 용량을 선택했을 때 효율적인 운전이 가능

# 정속형 방식

정속형은 정해진 출력으로 작동하다 목표 온도에 도달하면 멈추고, 온도가 올라가면 다시 같은 출력으로 가동하는 방식이다.

인버터보다 온도 변화가 크게 느껴질 수 있고, 재가동이 반복되면서 전력 사용이 증가할 수 있다.

# 구매 판단

최근 제품에서 단순히 “OO 인버터”라는 마케팅 명칭만 비교하기보다 동일한 시험 기준의 냉방효율, 실제 냉방능력, 사용 공간 적합성을 함께 확인한다.
