# 실제 면적을 모를 때의 간편 추정

실제 거실과 주방 면적을 정확히 알 수 있다면 그 면적으로 냉방능력을 계산하는 편이 더 정확하다. 도면이나 실측값이 없다면 주거 형태를 기준으로 대략적인 평형을 추정할 수 있다.

# 아파트와 빌라

일반적인 아파트와 빌라는 공급면적의 약 50% 수준을 거실용 에어컨 평형의 출발점으로 볼 수 있다.

이 기준은 대체로 다음 조건을 가정한다.

- 전용면적률이 약 70~80%인 구조
- 거실과 주방이 전용면적의 절반에 조금 못 미치는 구조
- 복층이나 비정형 구조가 아닌 일반적인 평면

## 발코니 확장

거실과 주방 쪽 발코니를 확장하면 실제 냉방할 공간이 커진다. 원문은 일반적인 확장 사례에서 한 단계, 약 2평 정도 큰 제품을 검토하는 방법을 제시한다.

확장 면적, 통창, 단열 상태가 서로 다르므로 단순히 “확장 여부”만으로 확정하지 않고 실제 구조를 함께 확인한다.

# 오피스텔과 단독주택

오피스텔과 단독주택은 공급면적 대비 실제 사용 면적의 비율이 일정하지 않으므로 공급면적보다 전용면적을 기준으로 판단하는 편이 낫다.

원문은 거실이 전체 실면적의 절반에 조금 못 미치는 일반적인 구조에서 전용면적의 약 65% 수준을 거실용 에어컨 평형의 간편 기준으로 제시한다.

# 간편 기준을 사용하면 안 되는 경우

다음과 같은 집은 비율 계산보다 실제 냉방 공간을 실측한다.

- 복층
- 거실이 비정상적으로 크거나 작은 구조
- 방과 거실의 경계가 거의 없는 개방형 구조
- 통창이 넓은 집
- 확장 범위가 큰 집
- 천장이 높은 집
- 단열 상태가 특이한 오래된 주택

# 질문에 답할 때의 사용 원칙

주거 형태 비율은 빠른 1차 추정에만 사용한다. 사용자가 정확한 거실·주방 면적을 제공하면 실면적 기준을 우선한다.
