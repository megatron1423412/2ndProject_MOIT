# 공기청정 기능을 볼 때 확인할 항목

에어컨에 `공기청정`이라는 문구가 있다고 해서 모두 같은 수준의 미세먼지 제거 성능을 제공하는 것은 아니다.

구매 시 다음을 구분한다.

- 공인된 공기청정 성능 인증 여부
- 인증 면적
- 미세먼지 필터 탑재 여부
- PM 센서 탑재 여부
- 오염도 표시 방식
- 냉방과 별도로 공기청정 단독 운전이 가능한지

# 인증과 적용 면적

원문은 CAC 인증을 받은 에어컨의 공기청정 성능을 인증 면적 안에서 평가할 수 있다고 설명한다.

공기청정 용도로 적극 사용할 계획이라면 실제 공간보다 여유 있는 인증 면적을 확인한다. 원문은 공기청정기 선택과 유사하게 실제 공간의 약 1.5배 수준을 예시로 든다.

예시:

- 실제 공간 5평
- 검토할 인증 면적 약 7.5평

이 수치는 일반 선택 원칙으로만 사용하고, 현재 인증 규정은 별도 확인한다.

# PM 센서

PM 센서는 실내 미세먼지 농도를 측정한다. 센서가 있으면 오염도에 따라 자동으로 운전하거나 현재 상태를 표시할 수 있다.

공기청정 단독 운전을 자주 사용할 사람에게는 센서가 편리할 수 있지만, 냉방만 필요한 사용자에게는 추가 비용 대비 가치가 낮을 수 있다.

# 오염도 표시

제품에 따라 다음 표시가 제공될 수 있다.

- 색상 단계로 청정도 표시
- 색상과 농도 숫자를 함께 표시
- PM1.0, PM2.5, PM10 등을 구분하여 표시

숫자 표시가 더 상세하지만 모든 사용자에게 필수는 아니다.

# 프리필터와 미세먼지 필터 구분

큰 먼지를 거르는 프리필터만 탑재하고도 넓은 의미의 공기청정 기능을 표시하는 제품이 있을 수 있다. 미세먼지 제거가 목적이라면 단순 기능명보다 실제 미세먼지 필터와 인증 여부를 확인한다.

# 구매 우선순위

이미 별도 공기청정기를 사용하고 있거나 공기청정 성능이 중요하지 않다면 고가의 공기청정 기능을 우선할 필요는 없다. 반대로 공간 절약과 단독 청정 운전을 중요하게 생각하면 검토할 가치가 있다.
