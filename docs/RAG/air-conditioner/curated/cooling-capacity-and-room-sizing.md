# 냉방능력과 평형의 의미

에어컨의 `냉방능력`은 소비전력과 별개의 값이며, 해당 제품이 어느 정도 면적을 효과적으로 냉방할 수 있는지를 나타낸다. 제품 설명에서 10평형, 16평형처럼 표현하는 값은 냉방능력을 소비자가 이해하기 쉬운 면적으로 환산한 것이다.

원문은 1평을 냉방하는 데 필요한 냉방능력을 약 400W로 설명한다. 예를 들어 10평 공간이라면 약 4,000W의 냉방능력을 기준으로 삼는 방식이다.

> 냉방능력 기준 예시: 사용 면적 10평 × 400W = 약 4,000W

이 계산은 제품의 소비전력을 구하는 식이 아니다. 냉방능력과 소비전력은 서로 다른 사양이므로 혼동하지 않는다.

# 거실과 침실의 선택 기준

## 거실과 주방이 연결된 공간

거실용 에어컨은 거실과 주방의 실제 면적을 합산한 뒤 약 1.3배의 여유를 두는 기준을 고려할 수 있다. 거실은 복도, 주방, 다른 방으로 냉기가 빠져나가기 쉽고, 출입과 채광의 영향도 크게 받기 때문이다.

다만 1.3배는 모든 집에 동일하게 적용되는 절대 규칙이 아니다. 거실 구조, 개방 정도, 단열, 창 면적, 층수, 채광에 따라 필요한 여유가 달라질 수 있다.

## 침실과 독립된 작은 방

침실처럼 문을 닫고 독립적으로 쓰는 작은 공간은 실제 사용 면적과 비슷한 평형을 우선 검토할 수 있다. 거실처럼 냉기가 여러 공간으로 크게 유출되지 않는 경우가 많기 때문이다.

# 1평 차이를 과도하게 고민하지 않기

16평형과 17평형처럼 1평 차이만 있는 두 제품은 실제 체감 냉방력 차이가 크지 않을 수 있다. 집의 실제 면적 역시 정확히 한 평 단위로 구분되기 어렵다.

1평 차이의 제품이 서로 다른 브랜드나 출시 시점이라면 평형 숫자만 비교하기보다 다음을 함께 본다.

- 실제 냉방능력
- 가격
- 필요한 부가기능
- 설치 조건
- 디자인과 크기
- 제품 데이터의 신뢰성

# 질문에 답할 때의 사용 원칙

이 문서는 “몇 평형이 적절한가”에 대한 초기 판단 근거로 사용한다. 실제 추천에서는 사용자가 입력한 공간 면적, 주거 형태, 채광, 단열, 개방 구조를 함께 반영한다.
