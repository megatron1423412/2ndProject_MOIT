# AI 자동모드의 역할

AI 또는 스마트 자동모드는 사용 패턴과 주변 환경을 바탕으로 목표 온도, 냉방 방식, 풍량, 풍향, 공기청정 등을 자동 조절하는 기능으로 소개된다.

# 기대할 수 있는 편의

- 자주 설정을 바꾸지 않아도 됨
- 사용 패턴에 맞춘 자동 운전
- 환경 센서와 연동한 풍량·풍향 조절
- 앱과 연동한 사용 정보 제공

# 만족도가 낮을 수 있는 이유

자동 판단이 사용자의 현재 선호와 다를 수 있다.

예를 들어:

- 취침 중 설정온도를 자동으로 올려 덥게 느껴짐
- 원하는 풍량이나 풍향과 다르게 작동
- 특정 순간에 원하는 모드를 자동으로 선택하지 않음
- 사용자가 직접 조절하는 것보다 복잡하게 느껴짐

# 구매 우선순위

AI 기능은 냉방능력, 공간 적합성, 가격, 필수 편의기능을 확인한 뒤 비교할 후순위 요소로 두는 편이 안전하다.

`AI`라는 명칭만으로 제품의 냉방성능이나 효율이 보장되는 것은 아니다. 실제로 어떤 센서와 자동 제어 항목을 지원하는지 제품별 사양을 확인한다.
