# 에어컨 편의·쾌적 기능과 관리

## 공기청정 기능은 인증과 적용 면적을 확인

에어컨의 공기청정 표기만으로 성능을 판단하지 않는다. 원본은 CAC 인증을 받은 제품은 공기청정 성능을 확인할 수 있다고 안내한다. 공기청정 기능을 주목적으로 쓸 경우에는 인증 여부, 공기청정 적용 면적, 오염도 감지용 PM 센서와 표시 기능을 확인한다.

원본은 공기청정 적용 면적을 실제 공간보다 여유 있게 보도록 제안한다. 다만 에어컨의 공기청정 기능이 독립 공기청정기를 완전히 대체하는지 여부는 실제 사용 공간, 운전 시간, 최신 인증 사양을 기준으로 판단한다.

## 바람 기능은 생활 불편을 기준으로 선택

제조사별 특수 바람 명칭 자체는 핵심 구매 기준으로 삼기 어렵다. 바람을 직접 맞는 것이 불편하면 간접 풍향 조절, 절전 운전처럼 실제로 원하는 사용 방식을 지원하는지 확인하는 편이 낫다. 빠른 냉방이 필요한 경우에는 급속 냉방 기능이 도움이 될 수 있지만, 명칭보다 냉방능력과 설치 환경이 우선이다.

원거리 냉방은 부엌처럼 거실에서 떨어진 공간까지 빨리 식히고 싶은 경우에 확인할 수 있는 기능이다. 집 구조와 문을 여는 습관에 따라 체감이 달라진다.

## 자동 모드와 원격 제어의 우선순위

AI·스마트 자동 모드는 사용 환경에 맞춰 운전을 조절하지만 모든 사용자에게 같은 만족도를 보장하지는 않는다. 원본은 이 기능만으로 상위 제품을 선택하기보다, 평형·효율·필요한 편의 기능을 먼저 고르라고 안내한다.

스마트폰 연동은 귀가 전 냉방, 상태 확인, 원격 조작이 필요한 사람에게 유용할 수 있다. 반면 원격 기능을 쓰지 않는다면 우선순위를 낮춘다.

## 자동건조와 풍향 조절

냉방 후 내부를 말리는 자동건조 기능은 곰팡이 관리에 도움이 되는 기능으로 원본에서 강조한다. 특히 벽걸이형처럼 기능 구성이 단순할 수 있는 제품은 자동건조 탑재 여부를 따로 확인한다. 풍향 조절은 바람이 사람에게 직접 닿는 상황이나 공간별 냉방 방향을 조정할 때 유용하다.
