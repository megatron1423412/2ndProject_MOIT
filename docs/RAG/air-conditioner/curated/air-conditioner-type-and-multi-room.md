# 벽걸이형과 스탠드형

벽걸이형이 거실에서 무조건 부족한 것은 아니다. 권장 면적에 맞는 냉방능력을 가진 제품이라면 거실에서도 사용할 수 있다.

## 벽걸이형의 장점

- 바닥 공간을 차지하지 않음
- 제품 가격이 상대적으로 낮을 수 있음
- 설치비가 상대적으로 낮을 수 있음
- 작은 공간이나 침실에 적합

## 벽걸이형에서 확인할 점

스탠드형보다 부가기능이 적은 모델이 많다. 특히 일부 제품은 자동건조, 좌우 자동 풍향, 고급 공기청정 기능 등이 없을 수 있으므로 필요한 기능을 개별 제품 DB에서 확인한다.

## 스탠드형의 장점과 한계

스탠드형은 넓은 거실에 적합한 고용량 제품과 부가기능이 많다. 반면 바닥 공간을 차지하고 구입비와 설치비가 커질 수 있다.

# 거실 한 대로 침실까지 냉방하기

큰 스탠드형 한 대를 거실에 설치해도 문과 복도 구조 때문에 침실까지 충분히 냉방하기 어려운 경우가 많다.

약한 냉방을 선호하거나 문을 열어두고 공기 순환을 보조하는 사용자는 어느 정도 만족할 수 있지만, 독립된 침실을 안정적으로 냉방하려면 별도 실내기가 더 적합하다.

# 2in1과 멀티형

거실과 침실을 각각 냉방하려면 스탠드형과 벽걸이형을 묶은 2in1 멀티형을 검토할 수 있다. 더 많은 방을 냉방해야 하면 3in1 등 다실 구성도 고려할 수 있다.

## 멀티형이 유리한 상황

- 거실과 침실을 서로 다른 시간에 사용
- 취침 시 작은 벽걸이 실내기만 가동하고 싶음
- 각 공간의 목표 온도를 독립적으로 조절하고 싶음
- 한 대의 대형 스탠드형을 계속 가동하는 것이 비효율적인 환경

## 확인할 점

- 실외기 하나가 여러 실내기를 지원하는 구성인지
- 동시 운전 시 용량 배분
- 배관 경로와 설치 가능 여부
- 실내기별 기능 차이
- 한 구성요소 고장 시 전체 사용에 미치는 영향

# 질문에 답할 때의 사용 원칙

제품 타입은 단순히 평형으로만 정하지 않는다. 냉방할 공간 수, 동시 사용 여부, 취침 패턴, 설치 공간과 예산을 함께 본다.
