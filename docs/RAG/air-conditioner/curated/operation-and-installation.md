# 에어컨 운전·설치 시 확인할 점

## 짧은 외출과 전원 관리

원본은 인버터 에어컨에서 약 2~3시간 이내의 외출이라면 실내 온도가 크게 다시 오르는 환경에서는 켜두는 편이 유리할 수 있다고 설명한다. 목표 온도에 도달한 뒤 낮은 출력으로 유지하는 특성 때문이다.

다만 실외 온도, 단열, 설정 온도, 부재 시간에 따라 결과는 달라진다. 장시간 외출이나 환기가 필요한 경우까지 같은 규칙을 적용하지 말고, 사용 환경과 제품의 최신 사용 설명서를 따른다.

## 제습 모드의 전력 사용

원본은 일반적인 제습 모드를 약한 냉방과 유사하게 설명한다. 제습이라는 이름만으로 항상 전력 사용이 적다고 단정할 수는 없다. 쾌적함을 위해 습도를 낮추는 것이 목적이면 제습 모드를 쓰되, 전력 절감을 우선한다면 실내 조건과 제품별 운전 방식을 확인한다.

## 설치 품질과 추가 작업 확인

설치 전에는 진공 작업 장비의 사용 여부를 확인한다. 원본은 매립 배관의 경우 배관 오염 가능성이 있어 질소 블로잉 같은 점검·처리의 필요성을 안내한다. 신축 주택이라도 배관 상태를 별도로 확인하는 것이 좋다.

타공, 배수 펌프, 철거, 운반 등은 설치 환경에 따라 추가 작업이 될 수 있다. 계약 전에 설치 범위와 추가 작업 조건을 확인하고, 현장 기사와 설치 위치·배수 경로·배관 상태를 명확히 협의한다.
