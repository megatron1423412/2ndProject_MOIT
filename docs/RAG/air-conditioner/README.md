# MOIT RAG curated package

This package currently contains a detailed air-conditioner curated Markdown set created from the user-provided NoSearch air-conditioner text.

The TV, refrigerator, and cordless-vacuum guides were not included because their saved HTML files were not attached to this conversation. Uploading the four RAG category folders as one ZIP is the safest way to create the remaining detailed sets from the exact saved source versions.
