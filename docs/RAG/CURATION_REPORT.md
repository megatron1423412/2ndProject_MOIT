## 공통 검증 및 삭제 결과

- 모든 큐레이션 문서는 완전한 YAML frontmatter, UTF-8 한국어 본문, 의미 있는 제목과 구매 질문에 답하는 본문을 포함한다.
- 본문에는 HTML·스크립트·스타일·탐색·광고 잔재가 없고, 문서 간 동일 본문이 없다.

- 이 보고서는 RAG 문서가 아니며, `curated` 디렉터리 밖에 둔다.
