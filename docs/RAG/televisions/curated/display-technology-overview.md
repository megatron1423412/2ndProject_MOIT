# 큰 분류: OLED와 LCD

## OLED

픽셀이 스스로 빛을 내고 끌 수 있는 자발광 방식이다. 별도 백라이트가 없다.

## LCD

패널 뒤의 LED 백라이트가 빛을 비춘다. 일반 LED, QLED·나노셀, Mini LED 계열이 모두 LCD 범주다.

# 일반 LED

LCD 패널과 일반 LED 백라이트를 사용한다.

## 장점

- 가장 저렴함
- 일반 방송과 OTT 시청에 충분한 화질
- 큰 화면을 낮은 예산으로 선택 가능

## 한계

- OLED와 Mini LED보다 블랙과 명암비가 약함
- 색재현력과 밝기가 상위 방식보다 낮을 수 있음

# 나노셀·QLED

일반 LCD에 특수 필름이나 코팅을 추가해 색 표현과 밝기를 높인다.

- 나노셀: 패널에 미세 입자 코팅
- QLED: 퀀텀닷 시트 사용

마케팅상 `QLED`라 불리지만 상용 제품 대부분은 자발광 QLED가 아니라 백라이트가 필요한 QD-LCD다.

# Mini LED 계열

백라이트 LED를 작게 만들고 구역별로 밝기를 제어해 LCD의 명암비를 개선한다.

- QNED·Neo QLED 등 브랜드 명칭 사용
- 같은 이름이어도 연식과 라인에 따라 Mini LED가 아닌 제품이 있을 수 있음
- 로컬디밍 구역 수와 제어 알고리즘이 중요

# 구매 원칙

기술 이름만 보지 않고 실제 패널, 백라이트, 로컬디밍, 밝기, 반사, 가격을 함께 본다.
