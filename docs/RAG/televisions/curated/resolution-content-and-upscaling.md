# 해상도의 의미

화면을 이루는 픽셀 수가 많을수록 같은 크기에서 더 세밀하게 표현할 수 있다.

- 4K UHD: 3840×2160
- 8K UHD: 7680×4320

8K는 4K보다 총 화소가 4배 많다.

# 하드웨어와 콘텐츠가 함께 필요

TV가 4K나 8K를 지원해도 입력 영상이 낮은 해상도면 원본 정보가 자동으로 생기지 않는다.

# 4K의 실용성

OTT와 유튜브에서 4K 콘텐츠가 널리 제공되어 4K는 일반적인 선택이 되었다.

# 8K의 한계

8K 콘텐츠가 적고 가격이 높아 일반 가정에서는 오버스펙이 될 수 있다. 매우 큰 화면, 가까운 거리, 전문 콘텐츠가 아니라면 체감 차이가 제한적이다.

# 방송 화질

케이블·IPTV·지상파의 실제 송출 해상도는 채널과 사업자에 따라 다르다. `UHD TV`라는 이유만으로 모든 방송을 4K로 보는 것은 아니다.

# 업스케일링

낮은 해상도 영상을 4K·8K 화면에 맞게 보정한다.

## 개선할 수 있는 요소

- 노이즈
- 계단현상
- 윤곽
- 질감
- 압축 흔적

## 한계

원본에 없는 세부 정보를 완전히 복원하지 못한다. 화질 엔진이 좋아도 저화질 방송이 실제 4K가 되는 것은 아니다.
