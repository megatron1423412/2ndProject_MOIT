# 일반 방송·OTT 중심 사용자

1. 시청거리와 화면 크기
2. 4K 해상도
3. 예산에 맞는 패널
4. 원하는 OTT와 OS
5. 설치 방식
6. HDR 호환
7. 필요한 포트
8. 사운드바 계획

일반 시청에서는 8K, 초고주사율, 고급 게임기능이 오버스펙일 수 있다.

# 영화 화질 중시

- OLED 또는 좋은 로컬디밍 Mini LED
- HDR 형식
- 블랙과 밝기
- 빛 반사
- 사운드바·eARC

# 스포츠

- 120Hz
- 모션 처리
- 밝기
- 큰 화면

# 콘솔 게임

- 4K 120Hz
- HDMI 2.1 실제 대역폭
- VRR·ALLM
- 입력지연
- 번인 사용 패턴

# 가성비 큰 화면

- 일반 LED 또는 보급형 QLED
- 4K
- 기본 HDR
- OS 반응과 A/S 확인

# 우선순위가 낮을 수 있는 사양

- 8K
- 사용하지 않을 음성비서
- 게임을 하지 않는데 고주사율
- 내장 스피커만을 위한 과도한 상위 라인
- 현재 콘텐츠가 지원하지 않는 고급 형식
