# 모델코드가 알려주는 정보

제조사 TV 모델명에는 다음 정보가 포함될 수 있다.

- 제품군
- 화면 크기
- 해상도
- 성능 라인
- 출시연도
- 판매지역·유통채널
- 디자인·스탠드·색상

# LG

- OLED 제품은 `OLED`로 시작하는 경우가 많음
- QNED·나노셀·울트라HD는 별도 제품군 코드 사용
- 숫자와 문자가 화면 크기, 등급, 연도, 판매처를 구분

같은 화면 크기라도 화질 엔진, HDR, 패널, 사운드, 부가기능이 달라질 수 있다.

# 삼성

- Neo QLED·QLED: `KQ`, `QN` 등으로 시작하는 경우
- UHD: `KU`, `UN` 등으로 시작하는 경우
- 뒤 숫자와 문자가 화면 크기, 연도, 라인, 판매처를 구분

# 주의

같은 규칙이 모든 연도와 국가에 적용되지 않는다. 모델코드는 후보를 빠르게 구분하는 도구이며 최종 사양은 공식 상세페이지와 MOIT DB를 확인한다.
