# 크기 표기

TV 인치는 화면 대각선 길이다. 1인치는 2.54cm다.

# 집 평수보다 시청거리

같은 평수라도 거실 구조, 소파 위치, TV장 깊이가 다르므로 눈에서 화면까지의 실제 거리를 기준으로 한다.

# 해상도와 크기

고해상도는 큰 화면이나 가까운 거리에서 장점이 잘 드러난다.

- 4K TV가 너무 작으면 FHD와 차이를 느끼기 어려움
- 낮은 해상도를 큰 화면에서 가까이 보면 픽셀이 보일 수 있음

원문은 4K, 시청거리 약 2m에서 65인치 이상도 픽셀이 깨져 보이지 않는 예를 제시한다. 이는 절대 기준이 아니라 참고치다.

# 큰 화면의 장점

TV는 며칠 사용하면 크기에 적응하는 경우가 많고, 영화와 스포츠 몰입감은 화면이 클수록 좋아질 수 있다.

# 너무 큰 화면의 한계

시청거리가 매우 짧으면 화면 전체를 한눈에 보기 어렵고 저해상도 콘텐츠의 결점이 크게 보인다. 화면 이동을 따라 눈과 목이 피로할 수 있다.

# 설치까지 고려

75인치 이상은 가격뿐 아니라 다음을 확인한다.

- 엘리베이터와 계단 반입
- 벽체 강도
- TV장 폭과 스탠드 간격
- 시청 높이
- 창문 반사
- 도어와 동선 간섭

# 판단 원칙

예산 내에서 한 단계 큰 화면과 한 단계 높은 패널 중 무엇을 우선할지 사용 목적에 맞춰 결정한다.
