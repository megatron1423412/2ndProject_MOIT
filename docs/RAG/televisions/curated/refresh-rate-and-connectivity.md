# TV 주사율과 외부기기 연결

## 일반 시청은 60Hz, 빠른 화면은 120Hz 검토

주사율은 TV가 1초에 표시할 수 있는 화면 수이며 Hz로 표기한다. 원본은 영화·드라마 같은 일반 영상은 60Hz로도 충분할 수 있다고 안내한다.

콘솔 게임이나 스포츠처럼 빠른 장면을 자주 본다면 120Hz 이상을 검토한다. 프레임 보간 기능은 부족한 프레임을 만들어 움직임을 더 부드럽게 보이게 할 수 있으나, 콘텐츠와 사용자 취향에 따라 체감이 다르다.

## 4K 고주사율에는 포트 사양도 필요

외부 게임기에서 4K 120Hz를 출력하려면 TV와 케이블, 게임기 모두 해당 신호를 처리할 수 있어야 한다. 원본은 HDMI 2.1 표기뿐 아니라 실제 대역폭도 확인하라고 주의한다. 포트 수와 어느 포트가 고대역폭을 지원하는지도 제품 사양에서 확인한다.

특히 고해상도·고주사율 게임이 주목적이면 화면의 주사율 숫자만 보지 말고, 입력 단자 사양과 게임기가 지원하는 출력 모드를 함께 맞춘다.
