# 스마트TV 기능

유튜브, OTT, 화면미러링, 음성검색, 앱 설치를 TV 자체에서 제공한다.

# IPTV 사용자의 판단

셋톱박스에서 OTT와 채널 기능을 모두 사용한다면 TV 스마트 기능의 중요도가 낮을 수 있다. 반대로 셋톱박스 없이 스트리밍만 볼 계획이라면 OS 반응속도와 앱 지원이 중요하다.

# 확인할 점

- 원하는 OTT 앱 지원
- OS 업데이트 기간
- 앱 실행속도
- 저장공간
- 스마트폰 미러링
- 리모컨 음성검색
- 광고와 추천 콘텐츠
- 개인정보 설정

# 소비전력

TV는 에어컨·건조기 같은 고전력 가전보다 소비전력이 낮지만 화면이 크고 밝기가 높을수록 증가한다.

OLED와 Mini LED, 80인치 이상 대형 화면을 오래 사용하면 차이가 누적될 수 있다.

# 표기값 구분

제조사는 최대, 정격, 평균, 대기전력을 다르게 표기할 수 있다. 실제 비교에서는 동일 기준의 소비전력과 하루 사용시간을 사용한다.

# 구매 판단

전기요금보다 패널·크기·가격이 우선인 경우가 많지만 하루 종일 켜두는 가정이나 상업공간은 반드시 비교한다.
