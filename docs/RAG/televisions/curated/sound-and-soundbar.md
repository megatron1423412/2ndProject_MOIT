# 채널과 출력

채널 표기의 소수점 앞은 일반 스피커, 뒤는 우퍼 수를 나타내는 경우가 많다. 출력 W가 높으면 큰 음량에서 여유가 있을 수 있다.

# 내장 스피커의 구조적 한계

TV가 얇아지면서 큰 스피커 공간을 확보하기 어렵고 소리가 화면 앞이 아니라 아래나 뒤로 향하는 제품이 많다.

같은 출력의 외장 스피커보다 대사 명료도와 저음이 부족할 수 있다.

# 내장 스피커를 우선하지 않아도 되는 경우

- 일반 방송과 뉴스 위주
- 작은 방
- 높은 음량을 사용하지 않음
- 별도 사운드바를 설치할 예정

# 사운드바가 유리한 경우

- 영화와 음악의 저음·공간감 중시
- 대사가 잘 안 들림
- 큰 거실
- 고가 TV의 화질에 맞는 음향을 원함

# 구매 전 연결 확인

- HDMI ARC/eARC
- 광출력
- TV와 사운드바 동시제어
- Dolby Atmos·DTS 지원
- 벽걸이 시 설치 공간

# 판단 원칙

사운드 때문에 상위 TV 라인으로 올리는 것보다 별도 사운드바가 비용 대비 효과가 클 수 있다.
