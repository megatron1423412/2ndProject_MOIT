# OLED

## 장점

- 픽셀을 완전히 끌 수 있어 깊은 블랙
- 매우 높은 명암비
- 넓은 시야각
- 백라이트가 없어 얇은 두께
- 어두운 영화와 HDR 콘텐츠에 강함

## 단점

- LCD보다 비쌈
- 장시간 고정 화면에서 번인 위험
- 밝기 보호를 위한 자동 밝기 제한이 장면 전환에서 느껴질 수 있음

# Mini LED LCD

## 장점

- 일반 LCD보다 높은 밝기와 명암비
- OLED 번인 부담이 적음
- 밝은 거실과 스포츠 시청에 유리할 수 있음

## 단점

- 로컬디밍 구역 주변에 빛 번짐이 생길 수 있음
- 상위 제품은 OLED만큼 비쌀 수 있음
- 같은 브랜드명 안에서도 성능 차이가 큼

# QLED·나노셀

일반 LED보다 색감과 밝기를 개선한 중간 단계다. OLED와 Mini LED의 높은 가격은 부담스럽지만 보급형보다 나은 색 표현을 원하는 사용자에게 적합하다.

# 일반 LED

화질 차이에 민감하지 않고 큰 화면과 가격을 우선한다면 충분하다.

# 사용자별 선택

- 최고 블랙과 영화 몰입: OLED
- 밝은 거실·번인 걱정·높은 밝기: Mini LED
- 중간 가격의 선명한 색: QLED·나노셀
- 일반 방송·OTT와 큰 화면 가성비: 일반 LED

# 매장 비교 주의

매장에서는 고사양 TV에 최적화된 데모 영상을 사용하므로 실제 방송과 화질 차이가 과장될 수 있다. 평소 볼 콘텐츠와 비슷한 영상으로 비교한다.
