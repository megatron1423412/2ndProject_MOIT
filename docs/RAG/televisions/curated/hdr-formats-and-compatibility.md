# HDR의 목적

HDR은 밝은 부분과 어두운 부분을 더 넓은 범위로 표현해 실제 눈으로 보는 것에 가까운 명암과 색을 보여주는 규격이다.

# 형식

## HDR10

가장 기본적인 범용 HDR 형식이다. 영상 전체에 공통 정보를 적용한다.

## HLG

방송 송출에 적합한 형식으로 실시간 방송에서 사용될 수 있다.

## HDR10+

장면별 메타데이터로 밝기와 색을 조정한다.

## Dolby Vision

장면별 메타데이터를 사용하며 많은 OTT와 디스크 콘텐츠에서 지원한다.

## Dolby Vision IQ

주변 밝기까지 반영해 Dolby Vision 영상을 조정하는 확장 기능이다.

# 콘텐츠와 TV가 같은 형식을 지원해야 한다

주로 사용하는 OTT, 콘솔, 블루레이가 어떤 HDR 형식을 제공하는지 확인한다. TV에 없는 형식은 기본 HDR이나 SDR로 재생될 수 있다.

# 형식만으로 화질이 보장되지 않음

HDR 로고가 있어도 패널 밝기, 블랙 표현, 로컬디밍, 색 영역이 부족하면 효과가 제한된다.

# 선택 원칙

- 일반 시청: HDR10·HLG 기본 지원이면 충분할 수 있음
- 영화와 OTT 화질 중시: Dolby Vision 또는 HDR10+ 확인
- 밝은 거실: 최대 밝기와 반사 처리 함께 확인
