# 빌트인의 핵심 장점

일반 냉장고는 깊이가 주방 가구장보다 깊어 앞으로 돌출되는 경우가 많다. 빌트인 냉장고는 깊이를 줄여 가구장과 비슷하게 맞추므로 주방 라인이 깔끔해진다.

# 핵심 단점

깊이를 줄인 만큼 같은 폭과 높이의 프리스탠딩 제품보다 용량이 작다. 가격은 크게 낮아지지 않아 용량 대비 가성비가 떨어질 수 있다.

인테리어보다 최대 용량과 가격을 우선하면 프리스탠딩이 유리하다.

# 4도어 빌트인

보통 600L대 초반으로 프리스탠딩 4도어보다 약 200L 이상 작을 수 있다. 3~4인 가구가 단독 사용하면 부족할 수 있어 김치냉장고나 1도어 제품을 추가하는 경우가 많다.

# 1도어 빌트인

세로로 긴 한 칸 구조이며 냉장전용, 냉동전용, 김치전용, 변온형 등으로 나뉜다.

## 장점

- 부족한 기능과 용량만 선택적으로 추가 가능
- 냉장·냉동·김치전용 조합을 자유롭게 설계
- 독특하고 정돈된 주방 구성 가능

## 단점

- 용량 대비 가격이 높음
- 여러 대를 조합하면 총비용과 전력, 설치 공간이 증가

# 2도어 빌트인

1도어와 비슷한 외형에 냉장실과 냉동실을 함께 제공한다. 작은 주방이나 1인 가구의 단독 냉장고, 기존 빌트인의 보조 용도로 적합하다.

# 설치 전 확인

- 가구장 실측 너비·높이·깊이
- 도어를 끝까지 열 수 있는 여유
- 통풍 공간과 콘센트
- 제품 반입 경로
- 여러 대 조합 시 문 방향과 높이 정렬
