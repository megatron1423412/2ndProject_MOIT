# 기본 구조와 용량

4도어 냉장고는 일반적으로 위쪽 전체가 냉장실, 아래쪽이 좌우 냉동실인 상냉장·하냉동 구조다. 대형 제품은 주로 800~900L대이며, 설치 공간이 작은 사용자를 위한 400~500L대 제품도 있다.

# 장점

## 좌우가 연결된 넓은 냉장실

냉장실 내부가 좌우로 막히지 않아 폭이 넓은 냄비, 반찬통, 수박 같은 식재료를 넣기 쉽다. 냉장실 전체가 상단에 있어 자주 쓰는 냉장 식품을 서서 꺼낼 수 있다.

## 큰 용량과 공간 효율

대형 냉장고 중 가장 큰 용량대를 선택하기 쉽다. 양문형과 외형 폭·깊이가 비슷하더라도 내부 용량이 더 큰 제품이 있을 수 있다.

## 최신 기능이 먼저 적용되는 경향

대중적인 주력 타입이어서 변온실, 투명도어, 직수형 아이스메이커, 대형 화면, 도어 색상 변경 같은 기능이 먼저 적용되는 경우가 많다.

# 단점

## 높은 가격

같은 800L대라도 양문형보다 가격이 높아질 수 있다. 대용량과 신기능이 필요하지 않다면 추가 비용의 실익을 따져야 한다.

## 하단 냉동실

냉동실이 모두 아래에 있어 식재료를 꺼낼 때 허리를 굽히거나 앉아야 한다. 냉동식품 사용이 잦거나 허리·무릎이 불편하면 양문형이 더 편할 수 있다.

# 적합한 사용자

- 3~4인 이상 가구
- 냉장실 사용 빈도가 높음
- 큰 식재료와 대형 용기를 자주 보관
- 최신 수납·제빙·정수 기능을 원함
- 예산보다 수납 편의와 용량을 우선
