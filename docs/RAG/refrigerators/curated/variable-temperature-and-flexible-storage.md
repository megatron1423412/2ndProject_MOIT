# 독립 변온실

별도 칸의 온도를 영하부터 영상까지 조절해 냉동·냉장·김치 보관 등으로 바꿔 쓰는 기능이다.

## 적합한 사용자

- 계절에 따라 냉장과 냉동 비율이 달라짐
- 김치 저장량이 많지 않아 별도 김치냉장고가 부담됨
- 육류·해산물을 짧게 보관할 독립 공간이 필요
- 대량 구매와 소분 보관을 자주 함

## 확인할 점

- 실제 조절 가능한 온도 범위
- 모드 전환에 걸리는 시간
- 칸 전체가 독립적으로 제어되는지
- 냄새와 수분이 다른 칸과 얼마나 분리되는지

# 변온 신선실

냉장실 하단의 일부 공간 온도를 식재료에 맞게 조절한다. 독립 변온실보다 범위가 좁지만 금방 사용할 육류, 주류, 신선식품 보관에 유용할 수 있다.

# 베버리지·디저트 공간

냉장실보다 조금 높은 온도로 와인과 디저트를 보관하는 프리미엄 공간도 있다. 특정 식품을 자주 보관하지 않는다면 우선순위는 낮다.

# 판단 원칙

기능 이름보다 실제 온도 범위, 공간 크기, 현재 식재료 보관 패턴을 본다. 사용하지 않을 칸 때문에 높은 가격을 지불하지 않는다.
