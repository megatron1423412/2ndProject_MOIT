# 투명도어

문을 열지 않고 내부를 볼 수 있도록 도어 일부가 투명해지는 기능이다.

## 장점

- 내부 확인을 위해 문을 여는 횟수를 줄일 수 있음
- 외관상 프리미엄 느낌
- 자주 찾는 식품 위치를 빠르게 확인

## 단점

- 가격 상승 폭이 큼
- 정리 상태가 그대로 보임
- 실제로 자주 확인하지 않으면 실용성이 낮음

# 색상 변경 패널과 엔터테인먼트 기능

LED 패널로 도어 색상을 바꾸거나 스피커·음성인식을 제공하는 제품이 있다. 편의보다 디자인과 재미에 가치를 두는 프리미엄 기능이다.

# 대형 화면과 카메라

냉장고 전면 화면으로 인터넷, 영상, 일정, 스마트홈 기능을 쓰고 내부 카메라로 식재료를 확인하는 제품도 있다.

## 적합한 사용자

- 냉장고를 스마트홈 허브로 적극 활용
- 주방에서 화면과 음성기능을 자주 사용
- 가격보다 새로운 경험을 우선

# 오토오픈과 오토클로징

- 오토오픈: 손이 자유롭지 않을 때 터치로 문을 열어줌
- 오토클로징: 문을 살짝 밀면 부드럽게 닫힘

손잡이 없는 빌트인 인테리어와 무거운 도어에서 편리할 수 있다.

# 앱 연동

앱에서 상태 확인, 온도 조절, 절전모드, 알림 등을 제공한다. 냉장고는 항상 작동하는 가전이므로 무선청소기보다 스마트 기능의 활용도가 높을 수 있다.

# 구매 판단

프리미엄 기능은 기본 냉장·수납 기능을 대체하지 않는다. 실제 사용 빈도와 추가 가격을 먼저 본다.
