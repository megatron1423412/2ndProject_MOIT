# 냉장고 수납 구조와 식재료 보관 기능

## 홈바와 자주 꺼내는 식재료

홈바는 전체 문을 열지 않고 음료나 자주 쓰는 식재료에 접근하는 공간이다. 원본은 편의성과 냉기 손실을 줄이는 점을 장점으로 들며, 4도어·양문형에서 가격 차이를 만드는 수납 기능 중 하나라고 설명한다. 홈바를 쓸 식재료가 정해져 있는지, 내부 용량 감소를 감수할 만한지 확인한다.

## 얼음 기능은 제빙 방식과 종류를 분리해 본다

아이스메이커는 자동 제빙과 수동 제빙, 그리고 원형·조각·작은 얼음 등으로 구분된다. 얼음을 자주 쓰는 가구는 필요한 얼음 형태, 급수·필터 관리 방식, 냉동실 공간의 변화까지 확인한다. 얼음 사용 빈도가 낮으면 복잡한 제빙 기능보다 기본 수납 공간이 더 유용할 수 있다.

## 변온실과 기본 수납 구성

변온실은 별도 칸의 온도를 조절해 식재료 성격에 맞게 보관하는 기능이다. 육류·채소·음료처럼 보관 목적이 자주 달라지는 가구에 적합할 수 있다. 변온 신선실의 온도 범위와 실제 사용할 식재료를 제품별로 확인한다.

접이식 선반은 큰 냄비나 높은 용기를 넣을 때, 도어 포켓은 병·소스류를 보관할 때 편리하다. 별도 수납 공간과 채소실 습도 조절도 식재료 보관 습관에 맞을 때 가치가 있다. 수납 기능은 많을수록 좋기보다 평소 장보기 양과 용기 크기에 맞아야 한다.
