# 구조

정수기 냉장고는 직수관, 정수 모듈, 출수 디스펜서를 냉장고 안에 통합한다. 주로 4도어·양문형과 일부 빌트인에 적용된다.

# 장점

## 주방 공간 절약

별도 정수기를 둘 필요가 없어 조리대와 선반 공간을 넓게 쓸 수 있다.

## 자동 얼음

각얼음, 원형 얼음, 디스펜서 얼음 등 자동 제빙 기능과 함께 제공되는 경우가 많다.

# 단점

## 높은 초기비용과 유지비

정수 모듈과 직수형 제빙 때문에 일반 제품보다 비싸고 필터 교체·관리 비용이 지속적으로 든다.

## 온수 미지원

냉수와 정수는 제공해도 일반적인 냉온정수기처럼 온수를 제공하지 않는 경우가 많다.

## 실사용 용량 감소

정수 모듈과 디스펜서가 냉장고 내부 공간을 차지한다.

# 적합한 사용자

- 별도 정수기를 둘 공간이 부족
- 얼음을 자주 사용
- 필터 관리 비용을 감수할 수 있음
- 온수가 필요하지 않음

# 구매 전 확인

- 직수관 설치 가능 여부
- 필터 가격과 교체 방식
- 방문관리 필수 여부
- 출수구와 제빙부 세척 방법
- 누수 보증과 서비스 정책
