# 기본 구조

양문형은 왼쪽 냉동실, 오른쪽 냉장실이 세로로 길게 나뉜 구조다. 용량은 대략 400L대부터 800L대 중반까지 폭넓다.

# 장점

## 냉장·냉동실 모두 서서 사용

두 칸 모두 위쪽 공간을 활용할 수 있어 자주 쓰는 냉동식품도 허리를 굽히지 않고 꺼낼 수 있다. 냉동실 사용이 잦거나 무거운 냉동 용기를 다루는 사용자에게 유리하다.

## 4도어보다 좋은 가성비

비슷한 대용량의 4도어보다 가격이 낮은 경우가 많다. 최신 프리미엄 기능보다 기본 냉장·냉동 기능과 용량을 중시하면 합리적이다.

# 단점

## 좁은 칸 폭

냉장실과 냉동실이 세로로 분리되어 각 칸의 폭이 좁다. 큰 냄비, 넓은 접시, 큰 과일을 보관하기 어렵고 안쪽 식재료가 가려질 수 있다.

## 신기능 적용이 느릴 수 있음

새로운 기능과 신제품은 4도어에 먼저 적용되는 경향이 있다. 다만 기본 냉각성능이 부족하다는 의미는 아니므로 신기능이 중요하지 않다면 큰 단점이 아닐 수 있다.

# 적합한 사용자

- 냉동실을 자주 사용
- 허리와 무릎 부담을 줄이고 싶음
- 대용량과 가격을 동시에 고려
- 넓은 냄비보다 세로 수납이 많은 편
