# 제균과 탈취를 구분하기

냉장고의 냄새를 줄이는 탈취와 미생물 억제를 목표로 하는 제균은 같은 기능이 아니다.

- `탈취`만 표기: 냄새 감소 중심
- `제균`, `UV`, `이온` 등이 표기: 제균 기능을 함께 주장하는 경우

두 기능이 모두 있고 가격 차이가 크지 않다면 실용성이 높을 수 있다. UV와 이온 방식 자체보다 필터 교체주기와 관리 방법을 확인한다.

# 추가 냉기 분사

도어 근처나 구석처럼 온도가 빨리 오르는 곳에 별도 냉기를 보내 문을 연 뒤 온도를 빠르게 낮추는 기능이다.

## 판단

가격 차이가 작다면 도움이 될 수 있지만, 이 기능만으로 수십만 원을 추가 지불할 정도의 체감 차이는 크지 않을 수 있다.

# 메탈·유리 소재

내부 벽이나 선반에 열전도율이 높은 소재를 사용해 냉기를 빠르게 전달하고 정온성을 보조한다.

적용 부위가 많을수록 가격이 높아질 수 있으나 실제 체감이 비례하지 않을 수 있다.

# 구매 원칙

냉각 기본성능, 도어 밀폐, 사용 습관이 우선이다. 부가기능은 가격 차이와 실제 식재료 보관 패턴을 함께 본다.
