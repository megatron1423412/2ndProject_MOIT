# 제품 무게보다 체감무게

전체 무게가 같아도 모터와 배터리 위치, 손잡이 각도, 사용자의 키에 따라 손목 부담이 다르다.

원문은 일반적인 체감 비율을 다음처럼 설명한다.

- 상중심: 전체 무게의 약 50~70%
- 하중심: 전체 무게의 약 20~40%

# 사용자의 키

키가 작으면 청소기와 바닥의 각도가 좁아져 상중심 본체 무게가 손목에 더 실릴 수 있다.

# 본체만 들고 쓰는 상황

차량, 계단, 문틀, 가구 위를 자주 청소하면 연장관과 메인 헤드를 뺀 본체 무게도 확인한다.

# 핸들링에 영향을 주는 요소

- 헤드 회전 반경
- 연장관 길이 조절
- 손잡이 굵기
- 무게중심
- 헤드가 바닥에 붙는 정도
- 낮은 가구 밑으로 눕힐 수 있는 각도
- 셀프 스탠딩 여부

# 구매 판단

손목이 약하거나 짧은 부분청소가 많다면 약간 낮은 흡입력보다 가벼운 제품이 만족도가 높을 수 있다.
