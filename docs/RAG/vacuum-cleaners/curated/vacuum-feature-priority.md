# 메인 청소기 우선순위

1. 중간 단계의 실제 흡입력
2. 마루·카펫에 맞는 헤드
3. 집 전체를 청소할 사용시간
4. 손목 체감무게
5. 배터리 교체와 소모품 공급
6. 세척과 머리카락 관리
7. 클린스테이션
8. 필요한 보조브러시

# 보조 청소기 우선순위

1. 가벼운 무게
2. 바로 꺼내기 쉬운 거치
3. 마루와 틈새 기본 성능
4. 낮은 가격
5. 간단한 세척
6. 과도한 액세서리 제외

# 우선순위가 낮을 수 있는 기능

## 스마트폰 연동

사람이 직접 들고 사용하는 제품이라 앱으로 상태를 확인하는 실익이 제한적일 수 있다.

## 소음 숫자 차이

대부분 큰 소음을 내므로 약간의 dB 차이보다 음색과 사용시간을 본다.

## 먼지통 최대용량

매우 작은 제품이 아니라면 자주 비우기 쉬운지가 더 중요하다.

# 사용자별 예외

- 반려동물: 털 브러시, 엉킴 방지, 필터
- 카펫: 카펫 헤드와 강한 중간단계 흡입력
- 손목 약함: 무게와 중심 우선
- 비염: 클린스테이션과 밀폐·필터
- 넓은 집: 배터리 2개와 긴 중간단계 사용시간
