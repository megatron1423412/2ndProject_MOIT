# 집 크기로 필요한 시간을 추정

원문은 가벼운 전체 청소시간을 대략 `평수 × 0.5분`으로 설명한다.

예시:

- 20평: 약 10분
- 30평: 약 15분
- 40평: 약 20분

가구가 많거나 꼼꼼히 청소하면 더 길어진다.

# 최대 사용시간보다 중간단계 확인

상세페이지의 최대 시간은 가장 약한 모드 기준인 경우가 많다. 최고출력은 5~10분에 끝날 수 있다.

메인 청소기는 실제로 많이 쓰는 중간 단계에서 집 전체 청소시간을 확보하는지가 중요하다.

# 탈부착 배터리

## 장점

- 수명이 줄면 사용자가 직접 교체
- 추가 배터리로 사용시간 연장
- 서비스센터에 본체를 맡길 필요 감소

## 일체형의 단점

교체를 위해 분해나 서비스가 필요하고 부품 단종 시 제품 수명이 짧아질 수 있다.

# 배터리 수명과 공급

원문은 리튬이온 배터리가 약 2~3년 또는 400~500회 충전 후 성능 저하가 시작될 수 있다고 설명한다. 실제 수명은 온도, 충전 습관, 출력 단계에 따라 다르다.

구매 전 다음을 확인한다.

- 교체 배터리 가격
- 별도 구매 가능 여부
- 제조사의 장기 공급
- 호환 배터리 안전성
- 2개 배터리 동시 충전 방식
