# 원터치와 트리거

## 원터치

한 번 누르면 계속 작동해 손가락 피로가 적다.

## 트리거

버튼을 계속 당겨야 하는 제품은 장시간 청소 시 불편할 수 있다. 최근에는 트리거 모양이어도 한 번 누르면 유지되는 제품이 있으므로 실제 작동 방식을 확인한다.

# 한 손 조작

청소기를 쥔 상태에서 엄지손가락으로 전원과 흡입력 단계를 조절할 수 있는지 본다. 버튼이 멀면 가구를 옮기면서 청소할 때 양손이 필요하다.

# 자동 흡입력 조절 방식

## 먼지량 감지

유입되는 먼지 양이 많을 때 출력을 높인다.

## 흡입압력 감지

흡입구 저항이 커지면 이물질이 많다고 판단한다.

## 바닥 센서

마루와 카펫을 구분해 출력을 바꾼다.

## 학습·배터리 관리

사용패턴이나 남은 배터리에 따라 출력을 조절한다.

## 복합 방식

브러시 저항, 압력, 센서를 함께 사용한다.

# 실익

사용자가 단계를 자주 바꾸지 않아도 되고 배터리를 아낄 수 있다. 다만 센서 판단이 취향과 다를 수 있으므로 수동 조절도 가능한지 확인한다.
