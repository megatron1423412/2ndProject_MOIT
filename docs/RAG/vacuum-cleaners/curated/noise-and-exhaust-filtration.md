# 소음

무선청소기는 강한 모터를 귀와 가까운 위치에서 사용하므로 조용한 제품을 찾기 어렵다. 원문은 많은 제품이 약 70~80dB 수준이라고 설명한다.

# 숫자보다 음색

같은 dB라도 날카로운 고주파가 더 거슬릴 수 있다. 리뷰에서 모터음의 음색, 최고단계 소음, 장시간 사용 피로를 확인한다.

# 헤드 마찰음

딱딱한 바퀴가 마루를 구르는 소리도 체감 소음과 층간소음에 영향을 준다.

# 배기필터

청소기는 먼지를 분리한 뒤 공기를 다시 실내로 내보낸다. 원문은 일반 가정에서 E11 이상을 권장한다.

필터 등급은 특정 크기 입자의 제거율을 기준으로 하며 등급 외에도 다음을 확인한다.

- 제조사가 제시한 입자 크기와 제거율
- 본체 밀폐 구조
- 필터 교체주기
- 세척 가능 여부
- 교체품 가격과 공급

# 알레르기 사용자의 판단

비염과 미세먼지에 민감하다면 높은 필터 성능뿐 아니라 먼지통 비움 방식과 본체 누설, 클린스테이션을 함께 본다.
