# 완전 분리 구조

먼지통과 필터가 본체에서 완전히 분리되어야 구석에 낀 미세먼지를 청소하기 쉽다.

# 물세척 가능 여부

먼지통과 금속·플라스틱 1차 필터는 물세척이 가능한 제품이 편리하다. 그러나 다음은 반드시 설명서를 확인한다.

- 최종 미세먼지 필터
- 전극이 내장된 먼지통
- 모터가 포함된 브러시
- 물걸레 급수부

세척 불가 부품을 물에 담그면 여과 성능 저하나 고장이 생길 수 있다.

# 건조

세척한 부품은 완전히 말린 뒤 조립한다. 습기가 남으면 냄새, 곰팡이, 모터 손상의 원인이 된다.

# 메인 브러시 관리

머리카락과 실이 롤러에 감기면 회전저항이 증가하고 청소성능이 떨어진다.

## 확인할 점

- 도구 없이 롤러 분리 가능
- 칼이나 가위가 들어갈 홈
- 롤러 물세척 가능 여부
- 교체 롤러 가격

# 엉킴 방지 구조

빗살형 스크래퍼가 회전 롤러의 머리카락을 모아 흡입구로 보내는 구조는 관리 시간을 줄일 수 있다. 반려동물과 긴 머리카락이 많은 집에서 유용하다.
