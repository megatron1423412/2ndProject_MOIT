# 무선청소기 배터리·무게·조작 편의

## 최대 시간이 아닌 실제 운전 시간 확인

최대 사용 시간은 가장 약한 단계 기준일 수 있다. 원본은 일상적으로 쓰는 중간 단계의 지속 시간을 확인하라고 안내한다. 집 전체를 청소하는 데 걸리는 시간과 강한 흡입을 쓰는 구간을 떠올린 뒤, 한 번 충전으로 충분한지 판단한다.

배터리는 탈부착 가능하면 교체와 충전에 편리하다. 추가 배터리를 살 수 있는지, 교체품 공급이 쉬운지도 장기 사용에 영향을 준다.

## 제품 무게보다 손목의 체감 무게

상중심 청소기는 제품 무게의 약 50~70%, 하중심은 약 20~40% 수준이 손목에 체감될 수 있다고 원본은 설명한다. 이는 구조와 사용 자세에 따른 참고치이며, 키가 작거나 손목 힘이 약하면 같은 제품도 더 무겁게 느낄 수 있다.

바닥 청소만 하면 헤드 지지로 부담이 줄 수 있지만, 창틀·문틈·차량처럼 본체를 들어야 하는 청소가 잦다면 본체 무게를 특히 확인한다.

## 조작 방식과 자동 흡입 조절

버튼을 계속 누를 필요 없이 한 번 눌러 작동하는 원터치 방식은 긴 청소에서 편할 수 있다. 자동 흡입 조절은 바닥 상태에 따라 출력을 바꾸는 기능이지만, 실제 체감과 배터리 사용 시간은 제품마다 다르다. 한 손으로 조작 가능한지, 헤드가 원하는 방향으로 잘 움직이는지도 실사용 편의에 중요하다.
